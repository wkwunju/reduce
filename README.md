# XTrack

XTrack monitors X (Twitter) accounts and delivers AI-generated summaries based on topics of interest.

## Highlights
- Scheduled monitoring (hourly, 6h, 12h, daily)
- Topic-focused AI summaries (Gemini/OpenAI)
- Email + Telegram delivery
- Execution history + summary storage
